Belajar Box Model

Box model adalah cara untuk menggambarkan elemen HTML sebagai kotak yang memiliki lebar, tinggi, margin, border, dan padding.

Dalam contoh ini, kita memiliki sebuah div dengan lebar 300px, margin 30px, border 5px solid merah, dan padding 25px.

Dengan menggunakan box model, kita dapat mengatur tata letak elemen-elemen di halaman web dengan lebih baik.
